🔥 Fire Date Predictor

Real-data wildfire date prediction for Thailand. NASA FIRMS · Hansen GFC · Open-Meteo ERA5.

Open dashboard → GitHub repo

What this is

A tree-ensemble regressor predicts days_until_fire (0–7) per 0.1° grid cell across Thailand. Predictions are rendered on a Leaflet map with urgency tiers calibrated against the held-out test split.

Data sources (real only)

  • NASA FIRMS VIIRS NRT — satellite hotspot detections
  • Open-Meteo Archive — ECMWF ERA5 daily reanalysis
  • Hansen Global Forest Change v1.11 — baseline canopy + recent loss
  • Thai province boundaries (CC BY 4.0)

No synthetic / simulated / interpolated values. If a real source isn't available the column is dropped, never faked.

Held-out test metrics

MAE ≈ 1.46 days · R² ≈ 0.28 · Acc±1day ≈ 52% · 16% better than the predict-mean baseline.

⚠ Disclaimer: This is a research / demo project, not an official emergency-response tool. Predictions are advisory and may not reflect current conditions. For authoritative wildfire information consult GISTDA's official disaster portal or local authorities.
